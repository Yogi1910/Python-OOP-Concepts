# Python-OOP-Concepts

Repository designed to explore and practice foundational object-oriented programming techniques in Python.

## Structure

- `basics/`: Classes, objects, and attribute/method fundamentals.
- `encapsulation/`: Examples of information hiding and validation.
- `inheritance/`: Single, multiple, and cooperative inheritance flows.
- `polymorphism/`: Method overriding and duck typing scenarios.
- `abstraction/`: Abstract base classes and Protocol-driven contracts.
- `design_patterns/`: Factory, strategy, and observer implementations.
- `solid_principles/`: Concise illustrations of the SOLID guidelines.
- `exercises/`: Practice prompts to reinforce each concept.
- `tests/`: Pytest specs showcasing how to verify OOP behavior.

## Getting Started

```bash
python basics/classes_and_objects.py
pytest
```
